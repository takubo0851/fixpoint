# fixpoint

apacheアクセス解析


overview

apache/logs/access.logからアクセスログを読み取り、
各一時間ごとのアクセス件数を表示し、
リモートホスト別のアクセス件数を件数の多い順に表示します。


description

アクセス件数の解析を行う期間を日付単位で指定できます。
デフォルトでは2017/11/20～2017/11/30に指定してあります。

ログファイルの解析は一行ごとに行い、そのつど変数を上書きしていきます。
